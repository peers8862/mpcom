**Pairs**<br>
Accounting tool for businesses and families built on [hledger](https://hledger.org). Track assets, liabilities, net worth, revenue, expenses, payroll, and more — all in plain text. <br>
Repo: [https://github.com/peers8862/pairs](https://github.com/peers8862/pairs)

**[Workwarrior](https://morgenpeers.com/projects/workwarrior)** <br>
Command line tool for personal productivity. A standardized syntax and set of scripts connects existing open source terminal tools and extensions, creating an all-in-one suite for work of all kinds. Now maintained by Babb. <br>
Repo: [https://github.com/babbworks/workwarrior](https://github.com/babbworks/workwarrior)

**Git-Game** <br>
Git-Game transforms learning Git from a chore into an adventure. Instead of dry tutorials, you'll complete themed missions that teach real Git commands through story-driven challenges. Set in a post-apocalyptic timeline where code repositories have been wiped out, you're the Operator tasked with restoring the Archive using Git. <br>
Repo: [https://github.com/peers8862/git-game](https://github.com/peers8862/git-game)

**Bookbuilder** <br>
Turns saved content from multiple sources into a searchable, AI-analysed knowledge base you can query, browse, export, and open in Obsidian. <br>
Repo: [https://github.com/peers8862/bookbuilder](https://github.com/peers8862/bookbuilder)

**XTool** <br>
Scrapes liked tweets and bookmarks from X into structured JSON. Combines live DOM scraping with intercepted GraphQL network responses to capture full text, expanded URLs, media, quote tweets, and author data. <br>
Repo: [https://github.com/peers8862/xtool](https://github.com/peers8862/xtool)

**ZPlus** <br>
A pip-installable toolkit for building Zensical sites whose content is structured data, not just prose. Doc types are declared once in a per-project zplus.toml manifest; zplus scaffolds the site, helps you (and your AI agents) fill it, links entries into a graph via typed ref fields, and generates dashboards, an action feed, and a machine-readable export on every build. <br>
Repo: [https://github.com/peers8862/zplus](https://github.com/peers8862/zplus)

**Thunderbird Migration** <br>
Copy your Thunderbird appearance, extensions, and settings from one machine to another — without touching mail, accounts, or passwords. <br>
Repo: [https://github.com/peers8862/thunderbird-limited-migration](https://github.com/peers8862/thunderbird-limited-migration)
